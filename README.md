# RikkiPatcher
A simple patch tool to modify the game '[鼓手余命十日谭 (Shiina Taki's Decameron)](https://store.steampowered.com/app/3269960)'

![screenshot](img/patcher.png)

## Features
- Extract and apply modified dialogue texts
